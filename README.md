# PeluqueriaProyectoFinalPOO 

Sistema de gestión para una peluquería en Java (JavaFX) con base de datos PostgreSQL, aplicando POO (herencia, clases abstractas, interfaces y patrón DAO).

##  Herramientas usadas

- Java 17+ / JavaFX (FXML + CSS)
- PostgreSQL + JDBC

---

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/usuario/PeluqueriaProyectoFinalPOO.git
   ```
2. Crear la base de datos y ejecutar `database/script.sql`.
3. Configurar credenciales en `db/Conexion.java`.
4. Ejecutar `Main.java` desde el IDE (o `mvn javafx:run`).
